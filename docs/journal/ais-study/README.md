# AIS Field Study — Session Materials

The working corpus of the AIS real-work study (2026-07): session briefs
(BRIEF-N), the analysts' own journals (JOURNAL-N), the vocabulary design
notes (VOCAB-NOTE v1/v2), the expert-channel digests, and rendered HTML
capsule reports per session (reports/).

Graded trial records live one level up in `docs/journal/trials/`; the
living capsule itself stays outside the repo
(/home/codex/ais-study/capsule.sqlite) along with regenerable run
artifacts (work/).

Shareability: scanned for credentials/personal data and passed the
capsule's export_preflight (scanner_clean) before committal, 2026-07-10;
human shareability review by James (the data owner) via this commit.
Vessel identifiers and positions are public AIS broadcast data; the
Enron report renders the public Enron corpus.
